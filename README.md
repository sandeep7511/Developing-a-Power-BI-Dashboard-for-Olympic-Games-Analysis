# Olympic Games Analysis: Power BI Dashboard

**Module:** SQL for Data Science  
**Assessment:** Assignment 1 (15% of total module mark) 

## Project Overview
This project aims to create an interactive Power BI dashboard that effectively visualizes historical data from previous Summer and Winter Olympic Games. The dashboard serves as a comprehensive analytical tool to interpret various aspects of Olympic data, uncover trends, evaluate country performances, and track key metrics across past events. 

## Data Source
The dataset for this project was extracted from the Wikipedia page: **"All-time Olympic Games medal table"**. 

The extracted data focuses on both Summer and Winter Olympics and includes the following key attributes:
* Country 
* Number of Summer/Winter Games participated 
* Gold medals 
* Silver medals 
* Bronze medals 
* Total medals 

## Data Processing
Prior to dashboard development, the raw web data was imported into Excel for cleaning and transformation. This stage involved handling missing values, standardizing inconsistent formatting, and restructuring the data into a schema optimized for Power BI ingestion.

## Dashboard Features
The interactive Power BI dashboard contains multiple user-friendly visualizations equipped with clear labels, titles, and legends. Key analytical views include:
* **Medal Breakdown:** Total medals won by each country comparing Summer vs. Winter Olympics.
* **Country Comparisons:** Visual comparison of overall medal counts across different nations.
* **Historical Trends:** Trend analysis over time for selected countries and overall global performance.
* **Medal Distribution:** A breakdown of gold, silver, and bronze medal distribution.
* **Efficiency Metrics:** Participation vs. success rate (comparing the number of games participated in against total medals won).

## Key Insights & Reporting
Alongside this dashboard, a 3-5 page PDF report has been compiled to document findings. The report answers key questions such as:
* Which countries dominate the Summer versus the Winter Olympics? 
* How have medal counts shifted across different Olympic eras? 
* What is the correlation between the frequency of a country's participation and their overall medal success?
